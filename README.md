# eslint-plugin

## Usage

Install package:

```bash
yarn add -D @fethcat/eslint-plugin
```

Add it to your package.json file:

```json
{
  "eslintConfig": {
    "extends": ["plugin:@fethcat/back"]
  }
}
```

Or in a frontend project

```json
{
  "eslintConfig": {
    "extends": ["plugin:@fethcat/front"]
  }
}
```
